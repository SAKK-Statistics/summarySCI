# summarySCI

Welcome to the Git repository of the package `summarySCI`.


## Installation 
You can install the package using


```
devtools::install_github(
  repo = "SAKK-Statistics/summarySCI",
  dependencies = TRUE
)
```

## Functions

Currently, the package contains three functions: 

- `summaryTable()`
- `summaryLevels()`
- `summaryByVisit()`
